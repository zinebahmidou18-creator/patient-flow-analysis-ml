# Patient Flow Analysis - CHU Lille

This project analyzes emergency department patient flow using real hospital data from CHU Lille.

## Dataset
- 335,000+ patient records
- 62 features
- Time-based and clinical variables

## Methods
- Data preprocessing and cleaning
- Feature engineering
- Machine learning models:
  - Gradient Boosting
  - MLP (Neural Network)

## Objectives
- Predict waiting time
- Analyze patient flow
- Improve hospital resource management

## Tools
- Python
- Pandas
- Scikit-learn
